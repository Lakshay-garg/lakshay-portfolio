Building Web Products End to End, From Requirement to Release

I'm an Angular developer with 3 years of experience shipping production web applications across healthcare, fintech, and enterprise domains. I work end to end — gathering requirements directly from clients, building responsive, reusable components, and owning delivery through to release. My core stack is Angular (v9–v20), TypeScript, RxJS, and Node.js / Express, with REST API integration at the center of most of what I build.

Modernization is a thread through my work: I've migrated four applications across major Angular and Node.js versions — v9 to v16 and v16 to v20 — refactoring legacy code to cut technical debt and lift performance. Alongside that, I've built and optimized a healthcare Progressive Web App with push notifications and an app-like install flow, delivered white-label solutions customized to client branding across multiple deployments, and integrated third-party and payment APIs including CIMS and Stripe.

I care about clean architecture and code that holds up long after it ships — which in practice means reusable components, real test coverage (70%+ with Jest on enterprise apps), and thorough code reviews in Agile/Scrum teams. I hold a B.Tech in Computer Science from The NorthCap University.
